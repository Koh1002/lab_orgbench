# ClaimSense AI ビジネス提案書（最終版）

## エグゼクティブサマリー

### 投資提案

日本の保険業界は、査定業務の非効率性、2026年保険業法改正への対応要件、急速な人材不足といった重大な構造的課題に直面しています。本提案は、ActuaryML社との共同開発による次世代型保険引受リスク評価プラットフォーム「**ClaimSense AI**」への投資機会です。

**シリーズA資金調達目標：12億円**

### 価値提案

ClaimSense AIは、以下の具体的な価値を提供します：

| 指標 | 実績値 |
|------|--------|
| 査定処理時間短縮 | 73%削減（45分→12分） |
| 月間業務削減時間 | 約140時間 |
| 顧客年間コスト削減 | 102万円/社 |
| 月額ライセンス料金 | 85,000円（競争力のある価格） |
| 3年後営業利益率 | 28% |
| 損益分岐点到達 | サービス開始後18ヶ月 |

### 財務見通し（3年累積）

- **初期投資**: 6,800万円以内
- **3年目営業利益**: 約8.4億円
- **顧客数目標**: 54社（初年度）→ 126社（3年目）
- **年間売上目標**: 初年度5.5億円 → 3年目12.9億円
